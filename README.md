一个 B 站直播抽奖工具，通过获取弹幕关键词进行抽奖，使用 Electron + React + Vite 构建

## 使用方法

在 Releases 中下载最新版本的安装包，安装并运行即可。

使用时在右上角登录 B 站账号，然后输入直播间号，点击开始获取即可。当结束获取后，点击开始抽奖即可。
